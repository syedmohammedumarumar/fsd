# **üìÇ Basic Linux/Unix Terminal Commands**

| No. | Command                           | Description                       |
| --- | --------------------------------- | --------------------------------- |
| 1   | `pwd`                             | Prints current working directory  |
| 2   | `ls`                              | Lists files and directories       |
| 3   | `cd [directory]`                  | Changes directory                 |
| 4   | `mkdir [dir_name]`                | Creates new directory             |
| 5   | `rmdir [dir_name]`                | Deletes an empty directory        |
| 6   | `touch [file_name]`               | Creates an empty file             |
| 7   | `rm [file_name]`                  | Deletes a file                    |
| 8   | `cp [source] [destination]`       | Copies a file or directory        |
| 9   | `mv [source] [destination]`       | Moves or renames file/directory   |
| 10  | `cat [file_name]`                 | Displays file content             |
| 11  | `nano [file_name]`                | Opens file in nano editor         |
| 12  | `echo "text" > file`              | Writes text into a file           |
| 13  | `man [command]`                   | Shows manual/help for command     |
| 14  | `history`                         | Displays previously used commands |
| 15  | `clear`                           | Clears terminal screen            |
| 16  | `whoami`                          | Displays current username         |
| 17  | `chmod +x [file_name]`            | Makes a file executable           |
| 18  | `ps`                              | Shows running processes           |
| 19  | `top`                             | Shows system resource usage       |
| 20  | `df -h`                           | Shows disk space usage            |
| 21  | `du -sh [folder]`                 | Shows folder size                 |
| 22  | `ifconfig` / `ip a`               | Shows IP address and network info |
| 23  | `ping [host]`                     | Tests network connectivity        |
| 24  | `grep 'text' [file]`              | Searches for text inside file     |
| 25  | `find . -name "*.txt"`            | Finds all `.txt` files            |
| 26  | `tar -czvf archive.tar.gz folder` | Compresses folder                 |
| 27  | `unzip archive.zip`               | Extracts ZIP file                 |
| 28  | `exit`                            | Closes terminal session           |
| 29  | `alias cls='clear'`               | Creates a command shortcut        |
| 30  | `uname -a`                        | Displays system information       |

# **üêô Git and GitHub Commands**

| No. | Command                                            | Description                      |
| --- | -------------------------------------------------- | -------------------------------- |
| 31  | `git --version`                                    | Checks installed Git version     |
| 32  | `git config --global user.name "Your Name"`        | Sets Git username                |
| 33  | `git config --global user.email "you@example.com"` | Sets Git email                   |
| 34  | `git init`                                         | Initializes a Git repository     |
| 35  | `git status`                                       | Shows repo status                |
| 36  | `git add filename`                                 | Stages a specific file           |
| 37  | `git add .`                                        | Stages all files                 |
| 38  | `git commit -m "message"`                          | Commits changes                  |
| 39  | `git log`                                          | Shows commit history             |
| 40  | `git remote add origin [URL]`                      | Adds GitHub remote               |
| 41  | `git branch`                                       | Lists branches                   |
| 42  | `git branch -M main`                               | Renames current branch to main   |
| 43  | `git push -u origin main`                          | Pushes code to GitHub            |
| 44  | `git pull`                                         | Pulls latest changes from GitHub |
| 45  | `git clone [URL]`                                  | Clones a repository              |
